Bootstrap

Bootstrap är ett av de äldsta och mest populära CSS-ramverken och erbjuder färdig CSS-kod och Javascript-funktionalitet. Ramverket innehåller funktioner som grid-system, responsiv design, fördefinierade stilar för text, förbättrat utseende av formulär och tabeller samt olika stilar för knappar. Det kan användas genom att ladda ned filerna direkt, installeras som ett npm-paket eller inkluderas via CDN.

Det har ett smidigt cheat-sheet som kan användas för för att underlätta utvecklingen, samt en mer omfattande dokumentation på Bootstraps egna webbplats med mer information om hur det används.

Mitt första intryck av Bootstrap är att det har många roliga funktioner så som accordions ("fäll ut-sektioner"), carousels (bildspel), modals (liknande pop up) och färdiga navbars (huvudmenyer). Det finns dessutom många fler roliga komponenter att använda sig av, bland annat spinners, cards, scrollspy och badges.

Utöver detta gillar jag det faktum att man får färdig kod för responsivitet, att man enkelt kan applicera klasser för att justera utseendet på olika komponenter och att det var enkelt att inkludera via CDN. Jag testade att installera det som npm-paket först men jag upplevde att det blev väldigt många filer att hålla reda på och filstrukturen överlag kändes lite mer komplicerad på det viset.

Jag hade gärna velat utforska Bootstrap lite mer för att förstå bättre hur det används och framför allt när det är fördelaktigt att använda det. Att man får mycket "gratis" med ett ramverk är väldigt tacksamt och kan snabba på utvecklingen, men jag får känslan av att man behöver vara mer påläst om det ramverket man ska använda för att faktiskt kunna utnyttja det fullt ut. Om man väl vet hur det fungerar ser jag att man nog kan vara väldigt kreativ med det.

Sammanfattningsvis kul att testa på ett större ramverk och testa sig fram med de olika komponenterna och klasserna som kommer på köpet!
